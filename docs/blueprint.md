# **App Name**: QPCTech

## Core Features:

- Service Summarization: AI-powered summarization of each IT service offered, using details from the QPCTech website.
- Service Recommendation Tool: A generative AI tool that helps users identify which IT services best suit their needs based on their input.
- Service Pages: Display service details like descriptions, benefits, and pricing.
- Contact Form: Include a contact form for users to inquire about specific services or request a consultation.

## Style Guidelines:

- Primary color: Deep blue (#046BD2) to convey trust and reliability. Inspired by the blue present in the QPCTech logo and the need for technological solutions.
- Background color: Light gray (#F5F5F5), providing a clean and modern backdrop that highlights content.
- Accent color: Vibrant cyan (#1c97d4), drawn from the blue highlights of the company's site design, is to be used for interactive elements and calls to action.
- Body font: 'Roboto' (sans-serif) for clear and accessible text, aligning with current website standards.
- Headline font: 'Barlow Semi Condensed' (sans-serif) for titles and headings to maintain brand consistency. Note: currently only Google Fonts are supported.
- Use flat design icons related to IT services, cybersecurity, and networking. The style should be consistent and professional.
- The layout should be clean and well-structured, using a grid system to ensure readability and ease of navigation. Ensure responsiveness across devices.
